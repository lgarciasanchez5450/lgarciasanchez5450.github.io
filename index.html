Description:

Project: Gravity-Planet Simulator
Platforms: Windows 10

Description:

This project made in conjunction with two classmates during junior year as a high school project. The project finished in 1st place for the Physics category. It can accurately reproduce Keplers second and third laws. It was an amazing experience to use programming skills to make a finished product for the first time.

Project: Somnï
Platforms: iOS 11.0 or later

Description:

Somnï is the third iOS release from DSA interactive, a group of high school students which I was a part of during my junior and senior years. This game was the year-long effort of a few students led by a mentor, all assets were made in-house and the game was built using the Unity Game Engine. As a part of the team I was responsible for implementing the interaction system which allows the player to interact with many other Game-Objects and be extensible for interoperability with the dialog system. I was also part of the team which implemented the levels by the design team into Unity.
